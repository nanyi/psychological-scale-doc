## 2. 概述

### 2.1 模块定位

用户与账户管理模块是心理测评系统的基础支撑模块，负责用户的注册、认证、权限管理、分组管理等功能，同时支持个人用户和企业用户两种类型，为系统其他模块提供用户基础服务。

### 2.2 功能摘要

本模块实现用户的注册登录、角色权限管理、用户分组、企业组织架构管理等功能，支持基于RBAC的权限控制模型，实现精细化的访问控制。

## 3. 业务需求

### 3.1 功能需求

#### FR-001: 用户注册

**功能描述**: 个人用户可注册成为系统用户

**输入**:
- 用户名（手机号/邮箱）
- 密码
- 验证码
- 昵称（可选）

**处理逻辑**:
1. 校验手机号/邮箱格式
2. 校验密码强度（8-20位，含大小写和数字）
3. 校验验证码正确性
4. 密码加密存储（BCrypt）
5. 创建用户基础信息
6. 初始化用户默认角色

**输出**: 注册结果

**业务规则**:
- 用户名全局唯一
- 密码加密存储
- 验证码有效期5分钟
- 未激活用户可保存24小时

#### FR-002: 用户登录

**功能描述**: 已注册用户可登录系统

**输入**:
- 用户名（手机号/邮箱）
- 密码
- 设备信息

**处理逻辑**:
1. 校验用户名和密码
2. 校验用户状态（未删除、未禁用）
3. 生成JWT Token
4. 记录登录日志
5. 返回用户信息和Token

**输出**: 登录结果、Token

**业务规则**:
- 密码错误5次锁定账户30分钟
- Token有效期7天
- 支持扫码登录

#### FR-003: 企业用户注册

**功能描述**: 企业管理员注册企业账号

**输入**:
- 企业名称
- 统一社会信用代码
- 联系人信息
- 管理员账号

**处理逻辑**:
1. 校验企业信息唯一性
2. 创建企业记录
3. 创建企业管理员账号
4. 初始化企业配额
5. 发送激活邮件

**输出**: 注册结果

**业务规则**:
- 企业名称全局唯一
- 统一社会信用代码全局唯一
- 管理员账号即为企业管理入口

#### FR-004: 角色管理

**功能描述**: 管理员可配置系统角色和权限

**输入**:
- 角色名称
- 角色描述
- 权限列表

**处理逻辑**:
1. 创建角色记录
2. 关联权限列表
3. 设置角色状态
4. 记录操作日志

**输出**: 角色配置结果

**业务规则**:
- 角色编码全局唯一
- 超级管理员角色不可删除
- 角色权限支持批量配置

**系统角色定义**:

| 角色编码 | 角色名称 | 说明 |
|----------|----------|------|
| SUPER_ADMIN | 超级管理员 | 系统最高权限 |
| ADMIN | 系统管理员 | 系统管理 |
| ENTERPRISE_ADMIN | 企业管理员 | 企业管理 |
| OPERATOR | 运营人员 | 运营管理 |
| PSYCHOLOGIST | 心理咨询师 | 专业分析 |
| USER | 普通用户 | 测评用户 |

#### FR-005: 权限管理

**功能描述**: 配置功能权限和数据权限

**输入**:
- 权限名称
- 权限编码
- 权限类型（功能/数据）
- 所属模块

**处理逻辑**:
1. 创建权限记录
2. 维护权限关系
3. 缓存权限数据

**输出**: 权限配置结果

**业务规则**:
- 权限编码全局唯一
- 支持父子权限层级
- 数据权限支持按组织/按用户

**权限数据结构**:

#### 3.3.2 用户登录接口

```json
{
  "username": "STRING|用户名(手机号/邮箱)",
  "password": "STRING|密码",
  "deviceInfo": "STRING|设备信息(可选)"

```

## 4. 性能需求

| 指标 | 要求 | 说明 |
|------|------|------|
| 登录响应 | ≤300ms | 登录响应时间 |
| 用户查询 | ≤200ms | 用户信息查询 |
| 权限校验 | ≤50ms | 接口权限校验 |
| 并发登录 | ≥500TPS | 支持并发登录 |

## 5. 可用性需求

- **SLA要求**: 用户接口可用性≥99.9%
- **Token刷新**: 支持自动刷新Token
- **单点登录**: 支持多端同时登录
- **账户锁定**: 异常登录自动锁定

## 6. 安全性需求

- **密码安全**: BCrypt加密存储
- **Token安全**: JWT + 签名验证
- **敏感操作**: 需要二次验证
- **登录保护**: 密码错误锁定机制
- **数据脱敏**: 返回数据脱敏处理
