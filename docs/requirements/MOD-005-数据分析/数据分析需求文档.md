## 2. 概述

### 2.1 模块定位

数据分析模块是心理测评系统的核心分析模块，负责数据统计、可视化展示、群体分析、常模对比等功能，为管理员、心理咨询师、企业HR等提供数据决策支持。

### 2.2 功能摘要

本模块实现数据驾驶舱、统计分析报表、群体趋势分析、常模数据对比、数据导出等功能，支持多维度的数据分析和可视化展示。

## 3. 业务需求

### 3.1 功能需求

#### FR-001: 数据驾驶舱

**功能描述**: 提供系统整体运营数据可视化展示

**输入**:
- 时间范围
- 用户角色权限

**处理逻辑**:
1. 汇总系统关键指标
2. 获取实时数据
3. 计算趋势数据
4. 渲染可视化图表

**输出**: 驾驶舱数据

**业务规则**:
- 数据实时更新
- 支持时间范围筛选
- 根据角色权限过滤数据

**驾驶舱指标定义**:

| 指标名称 | 指标说明 | 计算方式 |
|----------|----------|----------|
| 测评总量 | 累计完成的测评次数 | COUNT(测评记录) |
| 活跃用户数 | 日/周/月活跃用户 | COUNT(DISTINCT用户) |
| 量表使用排行 | 各量表使用次数 | COUNT(量表ID) |
| 测评完成率 | 完成率 | 完成数/开始数 |
| 平均测评时长 | 平均完成时间 | AVG(完成时间-开始时间) |
| 异常检出率 | 测评结果异常比例 | 异常数/总测评数 |

#### FR-002: 统计分析报表

**功能描述**: 提供多维度的统计分析报表

**输入**:
- 报表类型
- 筛选条件
- 时间范围

**处理逻辑**:
1. 解析报表配置
2. 查询原始数据
3. 执行统计计算
4. 生成报表数据

**输出**: 报表数据

**业务规则**:
- 支持多种报表类型
- 支持自定义筛选
- 报表数据可导出

**报表类型定义**:

| 报表类型 | 说明 |
|----------|------|
| 量表使用报表 | 各量表的使用统计 |
| 用户测评报表 | 用户测评行为统计 |
| 收入报表 | 订单和收入统计 |
| 测评结果报表 | 测评结果分布统计 |
| 企业使用报表 | 企业用户使用统计 |

#### FR-003: 群体趋势分析

**功能描述**: 分析不同群体的测评趋势和特征

**输入**:
- 群体维度（年龄/性别/职业/地区）
- 时间范围
- 量表范围

**处理逻辑**:
1. 按维度分组数据
2. 计算各组统计数据
3. 计算趋势变化
4. 识别异常群体

**输出**: 趋势分析结果

**业务规则**:
- 支持多个维度交叉分析
- 趋势数据按时间聚合
- 异常群体高亮显示

**群体维度定义**:

| 维度 | 说明 | 分组方式 |
|------|------|----------|
| 年龄 | 用户年龄 | 18-25, 26-35, 36-45, 46-55, 55+ |
| 性别 | 用户性别 | 男, 女, 未知 |
| 职业 | 用户职业 | 学生, 教师, 医护, 企业员工, 其他 |
| 地区 | 用户所在地区 | 省/市/区 |
| 企业 | 企业用户 | 按企业维度 |

#### FR-004: 常模对比分析

**功能描述**: 将个体测评结果与常模数据进行对比

**输入**:
- 用户ID
- 量表ID
- 常模组

**处理逻辑**:
1. 获取用户测评得分
2. 获取常模数据
3. 计算标准分（T分/Z分）
4. 生成对比结果

**输出**: 常模对比结果

**业务规则**:
- 常模数据按年龄性别分组
- 支持自定义常模
- 提供百分位排名

**常模数据结构**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | BIGINT | 常模ID |
| scale_id | BIGINT | 量表ID |
| dimension_id | BIGINT | 维度ID |
| group_type | VARCHAR(20) | 分组类型 |
| group_value | VARCHAR(50) | 分组值 |
| mean | DECIMAL(10,2) | 平均分 |
| std_dev | DECIMAL(10,2) | 标准差 |
| sample_size | INT | 样本量 |
| norm_source | VARCHAR(100) | 常模来源 |
| create_time | DATETIME | 创建时间 |

**计算公式**:

```
T分 = 50 + 10 × (原始分 - 常模均值) / 常模标准差
Z分 = (原始分 - 常模均值) / 常模标准差
百分位 = 正态分布累计概率(T分)
```

#### FR-005: 实时数据监控

**功能描述**: 实时监控测评进行状态

**输入**:
- 监控维度

**处理逻辑**:
1. 查询进行中的测评
2. 统计在线用户
3. 监控异常情况
4. 推送告警

**输出**: 实时监控数据

**业务规则**:
- 数据实时刷新
- 异常自动告警
- 支持大屏展示模式

**监控指标**:

| 指标 | 说明 |
|------|------|
| 当前在线 | 当前进行测评的用户数 |
| 进行中 | 各量表进行中的测评数 |
| 待处理 | 待审核/待处理任务数 |
| 异常告警 | 异常测评预警 |

#### FR-006: 数据导出

**功能描述**: 支持各类数据的导出

**输入**:
- 数据类型
- 筛选条件
- 导出格式

**处理逻辑**:
1. 校验导出权限
2. 查询数据
3. 生成导出文件
4. 返回下载链接

**输出**: 导出文件

**业务规则**:
- 导出权限严格控制
- 大数据量异步导出
- 导出文件保留7天

**导出格式支持**:

| 格式 | 说明 |
|------|------|
| Excel | .xlsx格式 |
| CSV | 逗号分隔文本 |
| PDF | 报表PDF |
